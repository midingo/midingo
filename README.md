# MIDINGO Game
A falling note style rhythm game that happens to also be a piano trainer.

## Getting started
MIDINGO is currently under development. There's not much here yet...

At the moment, only MIDI inputs will be supported. Computer keyboard support is somewhere on the roadmap.

## Documentation
The documentation repo for MIDINGO can be found at [midingo/docs](https://github.com/midingo/docs).

The docs cover:
- development (GDD)
- intended setup
- making charts
